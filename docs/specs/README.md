# qclaw-ui Specs

These documents version the current protocol, product direction, and implementation route inside the repository.

- `protocol-v1.md`: standard command, event, and state contract for qclaw-ui.
- `rfc-v1.md`: product and architecture baseline for the independent qclaw-ui workspace.
- `repo-bootstrap-v1.md`: repository structure, package responsibilities, and workspace conventions.
- `openclaw-integration-v1.md`: minimal integration path between qclaw-ui and openclaw/Electron.
- `implementation-map-v1.md`: execution order and code landing map for qclaw-ui integration work.

These files are derived from the planning documents created under the original workspace `steup/new` directory on 2026-03-26 and are now the in-repo source of truth for qclaw-ui work.
